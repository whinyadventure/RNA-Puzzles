Collective experiment for blind RNA structure prediction.

Aims of the project

  • To determine the capabilities and limitations of current methods for sequence-based 3D RNA structure prediction.
  • To find whether and how progress has been made, as well as what has yet to be done to achieve better solutions.
  • To identify whether there are specific bottlenecks that hold back the field.
  • To promote the available methods and guide potential users in the choice of suitable tools for real-world problems.
  • To encourage the RNA structure prediction community in their efforts to improve the current tools and to make automated prediction tools available.
  • To explore the underlying mechanism of ligand-RNA binding and the conformational changes in riboswitches.

Basic Rules

  • Confidential agreement: "I will use the provided sequence data for prediction purposes only. I will not distribute the sequences in any form outside my working group."
  • We strongly recommend that all the predictions are formatted in standard PDB format. It ensures that all the modellers have a fair comparison without considering the file format and prediction region. We provide a python script for generating standard PDB format files, only the coordinates need to be replaced by the predictors.
  • RNA chains should be named in alphabetical order. e.g. A for the first chain, B for the second chain.
  • Each group/server can submit up to 5 predicted models for one puzzle.
  • All models predicted for the single puzzle should be submitted in one PDB file like NMR format, i.e., every model should start with line "MODEL n" and end with the line "ENDMDL".
  • To submit, use our online submission system or send the file by email.

Call for participants

  • We are looking for crystallographers willing to provide us with a new crystal, cryo-EM or NMR structures. Any kind of RNA structure-related data is encouraged. Please, contact us for the details if you would like to share your RNA structure with us.
  • We encourage scientists interested in RNA 3D structure prediction to take part in RNA-Puzzles competition and push forward the field. If you want to join us, please, send us an email or register in the system.
